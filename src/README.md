# MediaScanner

Консольная утилита на C++17 для периодического сканирования заданного каталога в поисках мультимедийных файлов (аудио, видео, изображения) и формирования JSON-отчета.

## Особенности архитектуры
- Отсутствие внешних зависимостей (кроме подтягиваемого через CMake `nlohmann/json`).
- Элегантный обход дерева каталогов с обработкой `permission denied` (защита от падений на root-директориях в Linux).
- Кастомный асинхронный HTTP-сервер на raw-sockets, устойчивый к зависанию клиентов (`SO_RCVTIMEO`, `MSG_NOSIGNAL`).
- Потокобезопасность на базе `std::shared_mutex` (паттерн Readers-Writer lock) при доступе к JSON.
