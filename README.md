# MediaScanner

**MediaScanner** — это высокопроизводительная консольная утилита на C++17 для фонового сканирования файловой системы. Программа ищет медиафайлы (аудио, видео, изображения) в заданном каталоге и формирует структурированный JSON-отчет. 

Утилита может работать в двух режимах:
1. **File Mode (по умолчанию)** — периодически сохраняет результат в файл `media_files.json` в домашней директории.
2. **HTTP Mode** — поднимает собственный асинхронный веб-сервер и отдает JSON по REST API (`GET /media_files`).

## Требования к окружению

Проект написан с использованием современных стандартов и POSIX-системных вызовов.
* **ОС:** Linux или macOS
* **Компилятор:** GCC 8+ или Clang 7+ (с поддержкой C++17)
* **Сборка:** CMake 3.14 или новее
* **Зависимости:** Библиотека `nlohmann/json` (скачивается и линкуется автоматически силами CMake, ставить вручную не нужно).

## Сборка проекта

Склонируйте репозиторий и выполните стандартные команды CMake:

```bash
# 1. Создаем директорию для сборки
mkdir build
cd build

# 2. Генерируем Makefile (CMake сам скачает nlohmann/json)
cmake ..

# 3. Собираем проект (флаг -j ускоряет сборку, используя все ядра процессора)
make -j$(nproc)
```
После успешной сборки в папке `build` появится исполняемый файл `media_scanner`.

## Использование и флаги

Запуск утилиты осуществляется из терминала. Программа работает в фоновом режиме до тех пор, пока вы не остановите её комбинацией `Ctrl+C`.

### Аргументы командной строки:

| Флаг | Описание | Значение по умолчанию |
| :--- | :--- | :--- |
| `--path <путь>` | Абсолютный или относительный путь к директории для сканирования. | Домашняя папка пользователя (`$HOME`) |
| `--interval <сек>` | Интервал между сканированиями в секундах. | `60` |
| `--http` | Включает режим HTTP-сервера (вместо записи в файл на диске). | Выключен |
| `--port <порт>` | Порт, на котором будет работать HTTP-сервер (требует `--http`). | `1234` |

## Примеры использования

**1. Базовый запуск (все по умолчанию)**
Сканирует домашнюю директорию раз в 60 секунд. Результат сохраняется в `~/media_files.json`.
```bash
./media_scanner
```

**2. Сканирование конкретной папки с частым обновлением**
Мониторинг папки с фильмами каждые 10 секунд:
```bash
./media_scanner --path /mnt/storage/movies --interval 10
```

**3. Запуск в режиме HTTP-сервера**
Сканирует папку пользователя `alice` раз в 30 секунд. Поднимает сервер на порту `8080`.
```bash
./media_scanner --path /home/alice/Downloads --interval 30 --http --port 8080
```
*Как проверить работу сервера в соседнем терминале:*
```bash
curl http://localhost:8080/media_files
```

**4. Сканирование всего жесткого диска**
Благодаря грамотной обработке `Permission Denied`, сканер не упадет при попытке прочитать системные директории от имени обычного пользователя:
```bash
./media_scanner --path / --interval 300
```

## Формат выходных данных

Пример генерируемого JSON-отчета:

```json
{
  "audio": [
    "Music/track_01.mp3",
    "Downloads/podcast.m4a"
  ],
  "images": [
    "Pictures/vacation.jpg",
    "Projects/logo.png"
  ],
  "video": [
    "Videos/tutorial.mp4"
  ]
}
```
*(Пути всегда указываются относительно директории, переданной в `--path`)*
